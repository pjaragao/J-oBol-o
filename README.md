## 🇺🇸 English

**JãoBolão** is a modern, multiplatform (Web & Mobile) platform for creating and managing football betting pools. With a premium interface and fluid experience, the project offers a complete betting system, real-time rankings, and monetization through subscriptions.

### 🚀 Technologies

- **Frontend Web:** [Next.js 15](https://nextjs.org/) (App Router), Tailwind CSS, ShadCN UI.
- **Mobile:** [React Native](https://reactnative.dev/) with [Expo](https://expo.dev/), NativeWind (Tailwind for Native).
- **Backend (BaaS):** [Supabase](https://supabase.com/) (PostgreSQL, Auth, Edge Functions, Realtime, Storage).
- **Payments:** [Stripe](https://stripe.com/) (Web) and [RevenueCat](https://www.revenuecat.com/) (Mobile IAP).
- **Sports Data:** [Football-Data.org API](https://www.football-data.org/).
- **Management:** [Turborepo](https://turbo.build/repo) (Monorepo).

### ✨ Features

- **🔐 Full Authentication:** Login via Email, Social (Google/Apple), and Magic Link.
- **⚽ Pool Management:** Create private or public groups, define scoring rules, and invite friends.
- **📊 Betting & Rankings:** Intuitive predictions before matches, betting visibility after kickoff, and automated rankings.
- **💳 Subscription System:** Free, Premium, and Pro plans with different limits and features.
- **🔄 Auto Sync:** Results and matches updated in real-time via Edge Functions and external APIs.
- **📱 Multiplatform:** Consistent experience across Web, iOS, and Android.

### 🛠️ Getting Started

1. **Clone the repo:** `git clone https://github.com/pjaragao/J-oBol-o.git`
2. **Install dependencies:** `npm install`
3. **Run dev environment:** `npm run dev`

## 🇧🇷 Português

O **JãoBolão** é uma plataforma moderna e multiplataforma (Web e Mobile) para criação e gestão de bolões de futebol. Com uma interface premium e experiência fluida, o projeto oferece um sistema completo de apostas, rankings em tempo real e monetização através de assinaturas.

### 🚀 Tecnologias

- **Frontend Web:** [Next.js 15](https://nextjs.org/) (App Router), Tailwind CSS, ShadCN UI.
- **Mobile:** [React Native](https://reactnative.dev/) com [Expo](https://expo.dev/), NativeWind.
- **Backend (BaaS):** [Supabase](https://supabase.com/) (PostgreSQL, Auth, Edge Functions, Realtime).
- **Pagamentos:** [Stripe](https://stripe.com/) (Web) e [RevenueCat](https://www.revenuecat.com/) (Mobile).

### ✨ Funcionalidades

- **🔐 Autenticação:** Login via Email, Social (Google/Apple) e Magic Link.
- **⚽ Gestão de Bolões:** Crie grupos privados ou públicos e defina regras de pontuação.
- **📊 Apostas e Rankings:** Palpites intuitivos e ranking automatizado em tempo real.
- **💳 Assinaturas:** Planos Free, Premium e Pro integrados.
- **📱 Multiplataforma:** Experiência consistente entre Web, iOS e Android.

### 🛠️ Como Começar

1. **Clone o repositório:** `git clone https://github.com/pjaragao/J-oBol-o.git`
2. **Instale as dependências:** `npm install`
3. **Inicie o desenvolvimento:** `npm run dev`
